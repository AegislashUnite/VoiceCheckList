# VoiceCheckList

## Esta es la idea de mi proyecto:
> Una aplicación web que va a realizar un llamado a lista por reconocimiento de voz
> Si el estudiante dice "presente" este lo va a reconocer y lo va a marcar como una asistencia
> Si no esta, el profesor dira "ausente" y lo va a guardar como una inasistencia de ese estudiante
> Si el estudiante acumula 3 o mas faltas, la aplicación web mandara un mensaje al estudiante acerca de sus reiteradas faltas
> Manejara un registro semanal, solo mostrando el trayecto de esa semana, pero guardara los datos de todos los dias para no sattrar la interfaz con información
> De igual manera, el profesor podra ver todo el registro entrado a la informacion del estudiante del que quiera saber acerca de sus faltas
